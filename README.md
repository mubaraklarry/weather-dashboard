Weather Dashboard

Responsive weather app built with React, Tailwind CSS, and OpenWeatherMap API.

Features
- Search weather by city name
- Display temperature, condition, icon, humidity, wind speed
- Loading and error states
- Mobile-friendly design

Setup
1. Clone the repo
2. npm install
3. Create `.env` 
4. npm run dev
