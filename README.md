Weather Dashboard

Responsive weather app built with React, Tailwind CSS, and OpenWeatherMap API.

Features
- Search weather by city name  
- Display current temperature, weather condition, icon, humidity, wind speed, and feels-like temperature  
- Recent searches list (saved in localStorage – persists after refresh)  
- Loading spinner during API calls  
- User-friendly error messages (e.g., "City not found")  
- Clean, modern UI with glassmorphism-style cards and gradients  
- Mobile-friendly layout (responsive design)

Technologies Used
- React
- Tailwind CSS
- OpenWeatherMap API
- JavaScript
- LocalStorage 

Setup & Installation
1. Clone the repo
2. npm install
3. Create .env and add API key
4. npm run dev
